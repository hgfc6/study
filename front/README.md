# Frontend 学习总览

这个目录用于整理前端基础知识、语法示例和框架入门代码。

## 当前结构

| 模块 | 说明 | 入口 |
| --- | --- | --- |
| `html` | HTML 基础标签、文档结构、常见元素示例 | [html.md](./html/html.md) |
| `css` | CSS 基础语法、选择器、布局和样式示例 | [css.md](./css/css.md) |
| `javascript` | JavaScript 基础、函数、原型、模块、Web API | [javascript.md](./javascript/javascript.md) |
| `vue` | Vue 基础、组件、脚手架示例、TodoList/GitHub 搜索等案例 | [Vue.md](./vue/Vue.md) |

## 推荐学习顺序

1. HTML
2. CSS
3. JavaScript 基础
4. JavaScript 进阶
5. Web API
6. Vue

## 各模块建议关注的知识点

### HTML

1. 文档结构
2. 常用标签
3. 表单
4. 语义化
5. 音视频
6. 表格与列表

### CSS

1. 选择器优先级
2. 盒模型
3. 定位
4. `display`
5. Flex 布局
6. Grid 布局
7. 动画与过渡
8. 响应式设计

### JavaScript

1. 数据类型与变量
2. 数组、对象、字符串
3. 函数与作用域
4. `this`
5. 原型与原型链
6. 模块化
7. 异步编程
8. DOM / BOM
9. 浏览器存储

### Vue

1. 模板语法
2. 数据绑定
3. 事件处理
4. 计算属性与侦听器
5. 条件渲染与列表渲染
6. 组件通信
7. 生命周期
8. 动画与过渡
9. Vue Router
10. Vuex
