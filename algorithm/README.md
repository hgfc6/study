## 必补知识点

1. 时间复杂度与空间复杂度
   - `O(1)`
   - `O(log n)`
   - `O(n)`
   - `O(n log n)`
   - `O(n^2)`
2. 线性结构
   - 数组
   - 链表
   - 栈
   - 队列
3. 查找结构
   - 哈希表
   - 二叉搜索树
   - 堆
4. 常见排序
   - 冒泡排序
   - 插入排序
   - 选择排序
   - 快速排序
   - 归并排序
   - 堆排序
5. 经典算法思想
   - 递归
   - 分治
   - 动态规划
   - 贪心
   - 回溯
